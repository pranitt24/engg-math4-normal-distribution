# Normal Distribution Explorer
### Engineering Mathematics 4 | Mini Project

An interactive, cinematic educational tool designed to help engineering students visualize Gaussian distributions and solve area-under-the-curve probability problems in real-time.

---

## ✨ Core Highlights
* **Interactive Graph Explorer:** Real-time Bell Curve visualization using the Desmos API. Adjust Mean (μ) and Standard Deviation (σ) and watch the curve morph.
* **Cinematic Solver:** A "video-like" experience where mathematical steps are written out line-by-line using a terminal-style typewriter effect.
* **Live Z-Table Scanner:** An automated matrix lookup system that highlights the exact row and column intersection for your specific Z-score.
* **Dynamic Shading:** Watch the probability area being "painted" onto the graph as the calculations unfold.

---

## 📖 Theoretical Coverage
The application serves as a comprehensive digital textbook, covering:
* **Mathematical Definitions:** Understanding Continuous Random Variables (CRV).
* **The PDF Formula:** Visual representation of the Gaussian function.
* **Standard Normalization:** The process of converting $X$ values to $Z$-scores using $Z = \frac{X - \mu}{\sigma}$.
* **Empirical Rule:** Visualizing the 68-95-99.7% distribution rules.
* **Area Under Curve:** Calculating probabilities between bounds, left-tail, and right-tail.

---

## 🛠️ Tech Stack
* **Frontend Framework:** React.js (Vite)
* **Graphing Engine:** Desmos API
* **Styling:** Tailwind CSS (Modern UI/UX)
* **Animations:** JavaScript RequestAnimationFrame & CSS Transitions

## 🚀 Getting Started

### Prerequisites
* [Node.js](https://nodejs.org/) (Latest LTS version)

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/pranitt24/engg-math4-normal-distribution.git](https://github.com/pranitt24/engg-math4-normal-distribution.git)
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd engg-math4-normal-distribution
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
5.  Open your browser to the local address provided (usually `http://localhost:5173`).
